# [Taco Bot](https://telegram.dog/KS_TacoBot)

Taco Telegram bot to spread appreciation, celebrate, and have a little fun.

---

## Setup

Collect these env variables to deploy this bot:-

* `API_ID`
* `API_HASH`
* `BOT_TOKEN`
* `DATABASE_URL` - Get it from `mongodb.com`
